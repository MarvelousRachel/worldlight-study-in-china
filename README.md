# WorldLight Study in China (static site)

A simple, modern one-page website for a China scholarship / admissions support agency.

## Edit content

Open `index.html` and replace:
- Partner universities list
- Contact details (email/WhatsApp/office)
- Your business trust/policy lines

## Run locally

Use any static server.

### Option A (Python)

```zsh
python3 -m http.server 5173
```

Then open http://localhost:5173

### Option B (VS Code Live Server extension)

Open `index.html` and click **Go Live**.

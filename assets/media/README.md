# Media assets

This folder is where you’ll put **your real photos & videos**.

## Structure

- `placeholders/` – safe placeholders used while real media is not ready.
- `programs/` – media for program pages (e.g. `civil-engineering-1.jpg`).
- `universities/` – media for university pages (covers, campus shots, etc.).

## How to replace placeholders

1. Add images/videos in the right folder.
2. Update the page to point to your new file path.
3. Keep filenames simple (lowercase, hyphens).

## Recommended sizes

- Hero images: 1600×900 (or larger), JPG/WEBP.
- Gallery images: 1200×800.
- Videos: MP4 (H.264), ideally < 20MB each for fast loading.
